# dailythonkbot
A slack bot for displaying data from Yle teksti tv.

# API Keys and tokens
You will need two kinds of secrets: YLE API key and Slack bot tokens.
Yleisradio API key: https://developer.yle.fi/

Slack API stuff: https://api.slack.com/ Tthe bot is technically a Slack app, so you'll need to create one first. Not covered here.)
